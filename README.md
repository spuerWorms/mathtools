# 🧮 数学计算器 - Math Calculator

一个基于 Vue.js 开发的智能数学计算器，支持小学1-6年级数学学习场景。

## ✨ 功能特点

### 🎯 四大计算模式
- **口算模式**：支持加减乘除基础运算
- **综合计算**：处理复杂数学表达式
- **递等式计算**：分步展示计算过程，支持简便运算
- **方程求解**：支持一元一次方程求解

### 🧠 智能特性
- ✅ 浮点数精度处理，避免 `0.99+0.99+0.99 = 2.9699999999999998` 问题
- ✅ 半角/全角符号自动转换
- ✅ 表达式格式验证与错误提示
- ✅ 支持简便运算定律（减法性质、分配律等）
- ✅ 一键复制计算结果

### 📝 支持的运算
- 四则运算：`+` `-` `×` `÷`
- 括号运算：`()` `[]`
- 小数运算：支持小数点
- 方程：一元一次方程，如 `2x+5=15`、`80(x−0.5)=64x`

## 🚀 快速开始

### 安装与运行

```bash
# 克隆项目
git clone https://github.com/spuerWorms/mathtools.git

# 进入项目目录
cd mathtools

# 直接打开 index.html 即可使用
# 或使用本地服务器
python -m http.server 8000
# 然后访问 http://localhost:8000
```

## 📖 使用说明

### 口算模式
1. 在两个输入框中输入数字
2. 选择运算符（+、-、×、÷）
3. 点击等号按钮查看结果

### 综合计算
1. 在输入框中输入完整表达式，如 `(12−212−96)÷4`
2. 点击等号按钮自动计算

### 递等式计算
1. 在输入框中输入表达式
2. 系统自动分步展示计算过程

### 方程求解
1. 在输入框中输入方程，如 `80(x−0.5)=64x`
2. 自动求解并显示结果
